# Настройка Битрикс24

Файл для фиксации опорных данных по настройкам портала: что настроено, где это находится и какие места нужно проверять при последующих изменениях.

Записи группируются по разделам Битрикс24, а не по датам. Если настройка затрагивает несколько сущностей, она размещается в наиболее логичном функциональном разделе, а связанные места перечисляются внутри записи.

## CRM

### Отправка автоматических сообщений

Автоматическая отправка сообщений в мессенджеры настроена в нескольких местах CRM.

**Бизнес-процессы сделок:**

- Отправка контактам ссылок на форму отзыва.
- Отправка сообщения участникам ДТП о задержке получения документов из ГАИ.
- Отправка сообщения участникам ДТП о получении документов из ГАИ.
- Отправка ссылки на добавление отзыва в 2Гис.

**Бизнес-процессы контактов:**

- Отправка сообщения о правильном выборе.

**Бизнес-процессы лидов:**

- Отправка сообщения о рисках.

**Роботы сделок:**

- Воронка "Выплата по ОСАГО", стадия "Отправка контактов".

**Что важно помнить:**

- Автоотправка распределена между бизнес-процессами сделок, контактов, лидов и роботом в сделках.
- При изменении текстов сообщений, условий запуска или канала отправки нужно проверять все перечисленные места.
- Для проверки нужно провести тестовую сделку, контакт или лид через соответствующий сценарий и убедиться, что сообщение отправляется в мессенджер.
